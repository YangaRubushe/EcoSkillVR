# EcoSkillVR
**EcoSkillVR** is an immersive Virtual Reality platform designed to train users in eco-friendly technology skills. This repository contains **start-up code only**. The full project is not included here. For access to the complete application, please reach out for more information.
---
## Demo Video
Watch a short demo of EcoSkillVR [here](https://drive.google.com/file/d/1LZltTayM2WzNVZ8N4lKhlHEatSCGo9fJ/view).  
---
## Features (Preview)
- **PPE Room** – Explore and select proper personal protective equipment.  
- **Assembly Room** – Interactively assemble battery packs.  
- **Equipment Room** – Learn the tools and machines used in eco-friendly tech.  
- **Battery Pack Handling** – Hands-on VR training with battery modules.  
- **Quiz Module** – Test your knowledge and earn points.

## Getting Started
This code is intended as a **starting point** for EcoSkillVR. To run the VR scenes and interact with the application, Unity 2022.3.60f1 is recommended. Make sure to import required assets and packages.
